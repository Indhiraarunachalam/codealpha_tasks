# Motion Detector System

## Objective

To detect motion using a PIR sensor and activate an LED and buzzer.

## Components Used

* Arduino UNO
* PIR Motion Sensor
* LED
* Buzzer
* Breadboard
* Jumper Wires

## Working Principle

The PIR sensor detects movement. When motion is detected, the Arduino turns ON the LED and buzzer. When no motion is detected, both remain OFF.

## Files Included

* motion_detector.ino
* circuit.png
* output.png

## Output

The system successfully detects motion and provides visual and audio alerts.
