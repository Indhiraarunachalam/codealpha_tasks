# Temperature and Humidity Monitoring System

## Objective

To monitor temperature and humidity using Arduino.

## Components Used

* Arduino UNO
* TMP36 Temperature Sensor
* Potentiometer (Humidity Simulation)
* LCD 16x2
* Breadboard
* Jumper Wires

## Working Principle

The TMP36 sensor measures temperature, while the potentiometer is used to simulate humidity values. The Arduino reads both values and displays them on the LCD and Serial Monitor.

## Files Included

* temperature_humidity_monitoring.ino
* circuit.png
* output.png

## Output

Temperature and humidity values are continuously monitored and displayed.
